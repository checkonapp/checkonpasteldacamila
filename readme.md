## 📋 Checklists

- **Subgerentes** — Turno Manhã e Tarde/Noite
- **Gerência (Derik)** — Visão do Turno (4 blocos, 17 itens)
- **DP (Viviane)** — Diário, Semanal e Mensal

## ⚠️ Importante

Os dados ficam salvos no `localStorage` de cada navegador.
A Camila só vê os dados no dashboard se acessar do mesmo dispositivo/navegador onde os checklists foram preenchidos.

Migração para banco de dados (Supabase) planejada para próxima versão.
